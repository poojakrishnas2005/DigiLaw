# ⚖️ DigiLaw - Empowering Legal Access


DigiLaw is a mobile-first platform designed to simplify legal information and resources. This project was built exclusively using **Google Technologies** to ensure high scalability, zero-infrastructure maintenance, and seamless mobile delivery.
##  Live Demo: [https://digilaw-f9d0e.web.app](https://digilaw-f9d0e.web.app)
##  Built with Google Technologies
We leveraged the **Google Cloud Ecosystem** to build a robust, serverless application:

* **Google Firebase Hosting**: Used for secure, lightning-fast global delivery of our web application.
* **Google Firebase CLI**: Integrated into our development workflow for automated deployments.
* **Cloud Firestore**: (Planned) Real-time NoSQL database for managing legal case studies and user data.
* **Google Fonts & Material Design**: Utilized for a clean, professional, and accessible user interface.
* **Python (Google App Engine Ready)**: Our backend logic is structured to run seamlessly on Google’s managed environments.

##  Key Features (Google Powered)
* **Serverless Architecture**: Zero infrastructure management using Firebase.
* **Mobile-First Design**: Optimized for the mobile browser experience.
* **Real-time Capabilities**: Built on Google's high-speed global network.
* **Secure & Scalable**: SSL-encrypted by default through Google's automated provisioning.

### Local Setup
1. Clone the repository: `git clone https://github.com/poojakrishnas2005/DigiLaw.git`
2. Install Firebase Tools: `npm install -g firebase-tools`
3. Login to Google: `firebase login`
4. Deploy: `firebase deploy`
